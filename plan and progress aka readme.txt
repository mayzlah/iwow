План по написанию приложения по получения илвла из армори WoW
1) Изучить апи баттлнета, выяснить как задать интересующий запрос и что мы при этом получаем +
2) Изучить процесс http передачи данных в java
3) Написать скелет программы. В частности, там должны быть классы main, GetRequest, SendRequest, GetResponse, PrintResponse. Определить поля и методы классов. Все поля должны быть приватными.
4) Наполнить классы содержанием, скомпилировать и исправить грубые ошибки
5) Протестировать и исправить оставшиеся ошибки.
6) Радоваться.
Примечание – периодически постить прогресс на GitHub вместе с этим планом, отмечать в плане прогресс выполнения задачи.
